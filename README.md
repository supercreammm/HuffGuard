# 基於 Huffman 演算法與 XOR 字典加密之輕量化文本壓縮系統

本專案是一個基於 Huffman 演算法與 XOR 字典加密之輕量化文本壓縮系統。專為算力與記憶體受限的物聯網 (IoT) 邊緣設備所設計，在極低的硬體負載下，同時實現資料壓縮與基礎資安防護。

## 系統特色
* **跨語言架構**：底層 C++ 負責高吞吐量的高速位元運算，前端 Python 負責流暢的人機互動介面。
* **極致輕量化**：C++ 核心運作時峰值記憶體佔用小於 15 KB，完美適應 Cortex-M0+ 等微控制器環境。
* **O(1) 字典混淆**：有別於傳統 AES 全域加密，本系統僅對壓縮標頭進行 XOR 位元運算。
* **無回饋亂碼防禦**：捨棄密碼正確性驗證機制，遭遇自動化腳本暴力破解時將輸出無意義外星亂碼，從物理邏輯上癱瘓攻擊。

## 檔案結構說明
* `main.cpp` / `main.exe`：C++ 壓縮與加密核心引擎。
* `gui.py`：基於 CustomTkinter 開發的現代化暗黑風格管理者控制台。
* `requirements.txt`：Python 執行環境依賴套件清單。
* `test_logs/`：(建議建立此資料夾) 存放用於測試的 1MB, 5MB, 20MB 系統日誌檔。
